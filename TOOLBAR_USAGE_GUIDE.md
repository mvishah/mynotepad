## Troubleshooting

**Q: I can't see the toggle button**
- A: The toggle button (â–²) appears on the far right of the toolbar, only when a PDF is loaded.

**Q: The toolbar won't hide**
- A: Make sure you have a PDF loaded. The toggle only works in PDF view mode.

**Q: How do I get the toolbar back if I lost the floating button?**
- A: Press **Ctrl+B** at any time to toggle the toolbar back on.

**Q: Does hiding the toolbar affect functionality?**
- A: No! All features still work. The toolbar is just visually hidden to save space.
